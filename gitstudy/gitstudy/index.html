git 常用指令学习

git网页全教程点这

已有项目上传github空仓库点这

总结

  • git init - 初始化仓库。
  • git add . - 添加文件到暂存区。
  • git commit - 将暂存区内容添加到仓库中
  • git clone - 拷贝一份远程仓库，也就是下载一个项目
  • git reset - 回退版本
  • git rm - 将文件从暂存区和工作区中删除
  • git mv - 移动或重命名工作区文件夹

提交日志

  • git log - 查看提交历史

  • git reflog 查看版本号

  • git reset —hard 6fcfc89 回退到6fcfc89这个版本

  • git blame <文件名> 以列表形式查看指定文件的历史修改记录

远程操作

  • git remote 远程仓库操作 git remote add origin 远程库地址
  • git clone 仓库地址
  • git fetch 从远程获取代码库
  • git pull 下载远程代码并合并
  • git push 上传远程代码并合并

分支

  • git checkout -b dev - 创建并切换到dev这个分支
  • git branch - 查看当前分支
  • git branch -a 查看所有分支
  • git checkout - 分支名 切换分支
  • git merge 分支名 - 用于合并指定分支到当前分支上，默认快速合并
  • git merge –no-ff -m “注释” 分支名 带参数的合并
  • git branch -d 分支名 -删除分支

.ssh文件生成

要用git生成.ssh文件，你可以按照以下步骤：

  1. 打开Git Bash，输入**ssh-keygen -t rsa -C "你的邮箱"**，然后连续按三个回车。
  2. 你会看到一个提示**Enter file in which to save the key，后面是一个路径，这就是你的.ssh文件夹的位置1**。
  3. 在.ssh文件夹里面，你会找到两个文件：id_rsa和id_rsa.pub。前者是私钥，后者是公钥。
  4. 你可以把公钥文件的内容复制到需要的地方，比如GitHub或GitLab。
